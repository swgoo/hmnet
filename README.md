# python_docker_vscode

# Script 실행 방법
1. [docker](https://www.docker.com/get-started/) 설치
1. [vscode](https://code.visualstudio.com/Download) 설치
1. vscode 실행
1. [extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.vscode-remote-extensionpack) 설치
1. vscode에서 F1 > ">Dev Containers: Reopen in Container" 입력 > Enter > 완료될때까지 기다리기
1. [extension](https://marketplace.visualstudio.com/items?itemName=ms-python.python)설치
1. Script 폴더에 있는 스크립트 실행 > F1 > ">Python: Run Python File in terminal" 입력 > Enter
